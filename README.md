# Turing-IA: Catálogo de Exhibición Automotriz (Frontend)

Esta es la interfaz de usuario de la plataforma de exhibición automotriz. Desarrollada con **Next.js**, esta aplicación consume la API previamente realizada para ofrecer una experiencia fluida de visualización de catálogo, gestión de inventario y autenticación segura.

## Características
* **Dashboard Dinámico:** Visualización de vehículos con filtros avanzados.
* **Sistema de Autenticación:** Manejo de sesiones con JWT (Login/Register).
* **Vistas por Roles:**
    * **Público:** Visualización del catálogo y detalles técnicos.
    * **Administrador:** Panel de gestión para Crear, Editar y Eliminar (CRUD) vehículos y marcas.
* **Diseño Responsivo:** Optimizado para dispositivos móviles y escritorio utilizando Tailwind CSS.

---

## 🛠️ Stack Tecnológico
* **Framework:** Next.js 
* **Estilos:** Tailwind CSS
* **Consumo de API:** Axios 
* **Validación de Formularios:** React Hook Form

## Configuración local
###  1. Clonar el repositorio
```bash
git clone https://github.com/EmilianoMt/Frontend
cd Frontend
```

### 2. Instalar dependencias
```bash
npm install
```

### 3. Ejecutar el servidor de desarrollo
```bash
npm run dev
```

La aplicacion se mostrara en  [http://localhost:3000](http://localhost:3000).
